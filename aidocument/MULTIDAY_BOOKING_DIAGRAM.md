## Database Query Examples

### Get All Multi-Day Bookings

```sql
SELECT 
    b.id,
    b.hall_id,
    b.customer_name,
    b.status,
    COUNT(d.id) as total_dates,
    MIN(d.booking_date) as start_date,
    MAX(d.booking_date) as end_date
FROM wp_shb_bookings b
LEFT JOIN wp_shb_booking_dates d ON b.id = d.booking_id
WHERE b.booking_type = 'multiday'
GROUP BY b.id
ORDER BY start_date DESC;
```

### Get All Dates for a Booking

```sql
SELECT 
    d.booking_date,
    d.slot_id,
    s.label as slot_name,
    s.start_time,
    s.end_time
FROM wp_shb_booking_dates d
JOIN wp_shb_slots s ON d.slot_id = s.id
WHERE d.booking_id = 123
ORDER BY d.booking_date ASC;
```

### Check Availability for Multiple Dates

```sql
-- Check if dates are available for hall_id=1, slot_id=5
SELECT 
    DATE('2025-01-15') as check_date,
    (
        SELECT COUNT(*) 
        FROM wp_shb_bookings 
        WHERE hall_id = 1 
        AND booking_type = 'single'
        AND booking_date = '2025-01-15'
        AND slot_id = 5
        AND status != 'cancelled'
    ) as single_bookings,
    (
        SELECT COUNT(*)
        FROM wp_shb_booking_dates d
        JOIN wp_shb_bookings b ON d.booking_id = b.id
        WHERE b.hall_id = 1
        AND d.booking_date = '2025-01-15'
        AND d.slot_id = 5
        AND b.status != 'cancelled'
    ) as multiday_bookings
```
